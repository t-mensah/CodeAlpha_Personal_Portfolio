# CodeAlpha_Calculator App


## Project Overview
This project is a basic arithmetic calculator built using HTML, CSS, and JavaScript as part of the Frontend Development Internship at CodeAlpha. The goal of this task was to create a functional calculator interface with a user-friendly design.


## Features
The calculator includes all basic arithmetic operations and display handling as required by the internship task:
HTML Structure: Defines the calculator layout, buttons, and display screen.
CSS Styling: Provides a modern, dark-themed UI (inspired by the provided design image), including rounded buttons and a responsive grid layout.
JavaScript Logic: Handles all user input, performs real-time calculations (`+`, `-`, `*`, `÷`), manages the display, and clears the input when necessary.


## Task Requirements (Task 2)
✅ Create a basic calculator using HTML, CSS, and JavaScript.
✅ Include all arithmetic operations: `+`, `−`, `×`, `÷`.
✅ Design a user interface with buttons and display screen.
✅ Handle user input, clearing screen, and real-time result display.
⬜ Bonus: Add keyboard support and styling enhancements. (Styling enhancements included)


## Technologies Used
Technology      Purpose
HTML5           Project structure and semantic markup
CSS3            Styling, layout (CSS Grid), and visual design
JavaScript      Core functionality and logic for calculations
